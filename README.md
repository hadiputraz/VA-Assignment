## 📄 Assignment Overview

This repository contains my **Vulnerability Assessment assignment**.

The objective of this assignment is to understand the difference between:

* **Threat** – Anything that can cause harm to a system.
* **Vulnerability** – A weakness in a system that can be exploited by a threat.

---

## 📘 Topics Covered

* Cybersecurity fundamentals
* Threat vs Vulnerability
* Common security risks
* Real-world security scenarios

---

## 🎯 Purpose

This assignment helps me to understand how **threats exploit vulnerabilities** in systems and why identifying weaknesses is important in cybersecurity.
